# EESSI mailing list

EESSI has a low-frequency mailing list, used for sending announcements about the bi-monthly update meeting
and important updates or news.

To subscribe to the mailing list, send an email with the following body to listserv@list.rug.nl:

```
SUBSCRIBE EESSI Firstname Lastname
```

To unsubscribe from the mailing list, send an email with the following body to listserv@list.rug.nl:
```
UNSUBSCRIBE EESSI
```
