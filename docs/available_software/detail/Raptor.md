# Raptor


Set of parsers and serializers that generate Resource Description Framework
(RDF) triples by parsing syntaxes or serialize the triples into a syntax.

<small>homepage: </small><span class="software-link">[https://librdf.org/raptor/](https://librdf.org/raptor/)</span>
