# GDRCopy


A low-latency GPU memory copy library based on NVIDIA GPUDirect RDMA technology.

<small>homepage: </small><span class="software-link">[https://github.com/NVIDIA/gdrcopy](https://github.com/NVIDIA/gdrcopy)</span>
