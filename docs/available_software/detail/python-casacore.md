# python-casacore


Python-casacore is a set of Python bindings for casacore,
a c++ library used in radio astronomy. Python-casacore replaces the old pyrap.

<small>homepage: </small><span class="software-link">[https://casacore.github.io/python-casacore/#](https://casacore.github.io/python-casacore/#)</span>
