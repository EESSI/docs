# Siesta


SIESTA is both a method and its computer program implementation, to perform efficient electronic
structure calculations and ab initio molecular dynamics simulations of molecules and solids.

<small>homepage: </small><span class="software-link">[https://siesta-project.org/siesta/About/overview.html](https://siesta-project.org/siesta/About/overview.html)</span>
