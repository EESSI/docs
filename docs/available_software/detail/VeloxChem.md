# VeloxChem


VeloxChem [RLV+20] is a Python-based open source quantum chemistry software for
contemporary and future hardware architectures. It features interactive program access through
Jupyter notebooks as well as massively parallel calculations in high-performance
computing (HPC) environments.

<small>homepage: </small><span class="software-link">[https://veloxchem.org/docs/intro.html](https://veloxchem.org/docs/intro.html)</span>
