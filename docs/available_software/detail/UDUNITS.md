# UDUNITS


UDUNITS supports conversion of unit specifications between formatted and binary forms,
 arithmetic manipulation of units, and conversion of values between compatible scales of measurement.

<small>homepage: </small><span class="software-link">[https://www.unidata.ucar.edu/software/udunits/](https://www.unidata.ucar.edu/software/udunits/)</span>
