# Tombo


Tombo is a suite of tools primarily for the identification of modified nucleotides
 from raw nanopore sequencing data.

<small>homepage: </small><span class="software-link">[https://github.com/nanoporetech/tombo](https://github.com/nanoporetech/tombo)</span>
