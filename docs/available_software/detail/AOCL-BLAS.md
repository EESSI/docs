# AOCL-BLAS


AOCL-BLAS is AMD's optimized version of
                 BLAS targeted for AMD EPYC and Ryzen CPUs.

<small>homepage: </small><span class="software-link">[https://github.com/amd/blis](https://github.com/amd/blis)</span>
