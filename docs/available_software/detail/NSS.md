# NSS


Network Security Services (NSS) is a set of libraries designed to support cross-platform development
 of security-enabled client and server applications.

<small>homepage: </small><span class="software-link">[https://developer.mozilla.org/en-US/docs/Mozilla/Projects/NSS](https://developer.mozilla.org/en-US/docs/Mozilla/Projects/NSS)</span>
