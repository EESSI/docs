# libtommath



LibTomMath is a free open source portable number theoretic
multiple-precision integer (MPI) library written entirely in C.


<small>homepage: </small><span class="software-link">[https://www.libtom.net/LibTomMath/](https://www.libtom.net/LibTomMath/)</span>
