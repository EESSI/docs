# libxc


Libxc is a library of exchange-correlation functionals for density-functional theory.
 The aim is to provide a portable, well tested and reliable set of exchange and correlation functionals.

<small>homepage: </small><span class="software-link">[https://www.tddft.org/programs/libxc](https://www.tddft.org/programs/libxc)</span>
