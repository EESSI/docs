# acpype


A tool based in Python to use Antechamber to generate topologies for chemical compounds and
to interface with others python applications like CCPN and ARIA.

<small>homepage: </small><span class="software-link">[https://github.com/alanwilter/acpype](https://github.com/alanwilter/acpype)</span>
