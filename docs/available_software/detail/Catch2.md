# Catch2


A modern, C++-native, header-only,
 test framework for unit-tests, TDD and BDD
 - using C++11, C++14, C++17 and later


<small>homepage: </small><span class="software-link">[https://github.com/catchorg/Catch2](https://github.com/catchorg/Catch2)</span>
