# PyEVTK


EVTK (Export VTK) package allows exporting data to binary VTK files for visualization and
data analysis with any of the visualization packages that support VTK files

<small>homepage: </small><span class="software-link">[https://github.com/paulo-herrera/PyEVTK](https://github.com/paulo-herrera/PyEVTK)</span>
