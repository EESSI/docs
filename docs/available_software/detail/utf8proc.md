# utf8proc


utf8proc is a small, clean C library that provides Unicode normalization, case-folding, 
and other operations for data in the UTF-8 encoding.

<small>homepage: </small><span class="software-link">[https://github.com/JuliaStrings/utf8proc](https://github.com/JuliaStrings/utf8proc)</span>
