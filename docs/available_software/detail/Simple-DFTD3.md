# Simple-DFTD3


Reimplementation of the D3 dispersion correction. The s-dftd3 project aims to
provide a user-friendly and uniform interface to the D3 dispersion model and
for the calculation of DFT-D3 dispersion corrections.

<small>homepage: </small><span class="software-link">[https://dftd3.readthedocs.io](https://dftd3.readthedocs.io)</span>
