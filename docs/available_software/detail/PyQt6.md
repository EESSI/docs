# PyQt6


PyQt6 is a comprehensive set of Python bindings for Qt v6. It is implemented as more than 35 extension
modules and enables Python to be used as an alternative application development language to C++.
This bundle includes PyQtWebEngine, a set of Python bindings for The Qt Company’s Qt WebEngine framework.
