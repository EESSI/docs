# libglvnd


libglvnd is a vendor-neutral dispatch layer for arbitrating OpenGL API calls between multiple vendors.

<small>homepage: </small><span class="software-link">[https://gitlab.freedesktop.org/glvnd/libglvnd](https://gitlab.freedesktop.org/glvnd/libglvnd)</span>
